This is simple console based c program made for cyber cafe.
It helps the owner manage customer usage time and calculate the bill automatically.
Instead of writing customer details and time records on paper, the owner can use this program to start and end customer sessions through a simple menu.
The program uses the time.h library to calculate how long a customer used the computer and then generates the total bill based on the usage time.
